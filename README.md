# Projecto Test — «Какой вы руководитель?»

Небольшое фронтенд‑приложение: одностраничный тест из 7 вопросов, который определяет управленческий стиль и отправляет результат на почту через EmailJS.

## Что внутри

- **Экраны**:
  - приветствие
  - вопросы
  - ввод email
  - экран успеха
- **Логика**:
  - ответы сохраняются в состоянии
  - результат вычисляется по максимальному числу попаданий в один из типов: `Авторитарный`, `Контролер`, `Стратег`, `Делегатор`, `Наставник`
  - при вводе email результат отправляется через EmailJS 

## Структура проекта

```
.
├── index.html                 # основная страница приложения
├── mail.html                  # HTML-шаблон письма (для EmailJS)

```
